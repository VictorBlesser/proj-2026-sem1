Adicionar padrão

Para adicionar um padrão customizado ao sistema, clique no botão PADRÕES CUSTOMIZADOS, preencha apenas o primeiro formulário, preenchendo o campo Nome:, dando um nome para o novo padrão. Após isso, preenchendo o campo Especificação do Novo Padrão: com as especificações do padrão, contendo informações dos tipos baseados no padrão. Por fim, clique em ADICIONAR, para confirmar a adição do padrão. Segue abaixo a estrutura que deve ser usada para configurar as especificações de um padrão customizado:


Estrutura de Especificação de novo padrão: 

"gramatura": 95, "cor": "white", "tipos": [ {"Meio_A0": "420.5 x 594.5 mm"}, {"Meio_A1": "297 x 420.5 mm"}, {"Meio_A2": "210 x 297 mm"} ]
