Introdução

Este é o manual do SAIP, contendo todas as funcionalidades e os modos de operação, com exemplos práticos de uso. Por favor, siga as instruções passo-a-passo corretamente para fazer a gestão da produção e configurar o programa de maneira adequada. É altamente recomendado que se leia também a especificação ISO 216, que não vem inclusa neste material, pois o conteúdo cobre aspectos sobre formatos de papel previstos e suas regras de padronização.

Adicionar pedido

Para adicionar um novo pedido ao sistema, clique no botão GERIR PEDIDOS, selecione o tipo de impressão desejado no campo Tipo:. Em seguida, informe a quantidade no campo Quantidade:, utilizando um valor entre 1 e 1000. Após isso, selecione a gramatura desejada no campo Gramatura: e escolha a cor de impressão na seção Cor de impressão:. Depois de preencher todas as informações do pedido, clique em ENVIAR para confirmar o envio do pedido ao sistema.

Cancelar pedido

Para cancelar uma operação de pedido, acesse a tela inicial do sistema e clique no botão GERIR PEDIDOS. Em seguida, será exibida a tela de gerenciamento de pedidos, contendo os campos de configuração do pedido e os botões de ação do sistema. Caso deseje interromper a operação atual, clique no botão PARAR para cancelar o processo do pedido.

Adicionar padrão

Estrutura de Especificação de novo padrão: 

"gramatura": 95, "cor": "white", "tipos": [ {"Meio_A0": "420.5 x 594.5 mm"}, {"Meio_A1": "297 x 420.5 mm"}, {"Meio_A2": "210 x 297 mm"} ]

Remover padrão

Para remover um padrão customizado do sistema, clique no botão PADRÕES CUSTOMIZADOS. Em seguida, localize o campo Selecione o Padrão Customizado para Remover: e informe o título do padrão customizado que deseja excluir. Após selecionar o padrão desejado, clique em REMOVER PADRÃO CUSTOMIZADO para confirmar a remoção do padrão do sistema.
